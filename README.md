# Git and GitHub Lab Project

## Project Description
This project demonstrates basic Git and GitHub workflow including:
- Creating and managing files
- Staging and committing changes
- Branching and merging
- Pushing to remote repository

## Files Included
1. Main.java - Java program with greeting
2. script.py - Python script
3. styles.css - CSS stylesheet
4. index.html - HTML webpage
5. app.js - JavaScript file
6. data.json - Sample JSON data
7. README.md - This documentation file

## How to Run
Each file can be executed or opened in its respective environment.
